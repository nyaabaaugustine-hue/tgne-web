## 🚀 INSTALLATION STEPS

### Step 1: Upload Theme to WordPress

1. Log in to your WordPress admin dashboard
2. Go to **Appearance → Themes**
3. Click **Add New** button
4. Click **Upload Theme** button
5. Click **Choose File** and select `creote-theme-clean.zip`
6. Click **Install Now**
7. After installation, click **Activate**

### Step 2: Install Required Plugins

The theme will prompt you to install required plugins:

**Required Plugins:**
- Contact Form 7
- One Click Demo Import
- Meta Box
- Revolution Slider (included in package)
- Creote Addons (included in package)

**Optional Plugins:**
- WooCommerce (for e-commerce)
- Mailchimp for WP
- YITH WooCommerce Compare

**Plugin Installation:**
1. Go to **Appearance → Install Plugins**
2. Select all required plugins
3. Click **Install** then **Activate**

### Step 3: Import Demo Content (Optional)

1. Go to **Appearance → Import Demo Data**
2. Choose your preferred demo version:
   - Elementor Version 1
   - Elementor Version 2
   - WPBakery Version
3. Click **Import** and wait for completion

## ⚙️ THEME CONFIGURATION

### After Activation:

1. **Set Permalinks:**
   - Go to **Settings → Permalinks**
   - Select **Post name**
   - Click **Save Changes**

2. **Configure Theme Options:**
   - Go to **Appearance → Theme Options**
   - Configure colors, typography, layout settings

3. **Create Menus:**
   - Go to **Appearance → Menus**
   - Create your navigation menu
   - Assign to "Primary Menu" location

4. **Set Homepage:**
   - Go to **Settings → Reading**
   - Select "A static page"
   - Choose your homepage

## 🆘 TROUBLESHOOTING

### If theme doesn't activate:
1. Check PHP version (requires 7.4+)
2. Check WordPress version (requires 6.0+)
3. Increase PHP memory limit to 256MB
4. Check file permissions

### If demo import fails:
1. Increase PHP max_execution_time to 300
2. Increase PHP memory_limit to 256M
3. Install Revolution Slider plugin first
4. Import sliders manually if needed

### If plugins won't install:
1. Upload plugins manually via **Plugins → Add New → Upload Plugin**
2. Use plugin files from `creote-package-v-2-9-0/pluigns/` folder
