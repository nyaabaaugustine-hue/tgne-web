# TGNE Web

TGNE (The Great Network Enterprise) - Corporate Website

## Project Overview

This repository contains the website implementation for TGNE using the Creote WordPress theme.

## Contents

- `website-implementation/` - Implementation guides and documentation
- `creote-READY-TO-UPLOAD.zip` - WordPress theme ready for deployment
- Theme customization documentation

## Theme Information

- **Theme:** Creote v2.9.0
- **Type:** Business/Consulting WordPress Theme
- **Status:** Verification removed, ready to use

## Installation

1. Upload `creote-READY-TO-UPLOAD.zip` to WordPress
2. Go to Appearance → Themes → Add New → Upload Theme
3. Install and activate the theme
4. Install required plugins when prompted

## Documentation

See `website-implementation/` folder for:
- Implementation guide
- Brand guidelines
- Sitemap
- Configuration details

## Requirements

- WordPress 6.0+
- PHP 7.4+
- MySQL 5.6+
